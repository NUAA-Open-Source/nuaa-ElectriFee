# nuaa-ElectriFee

## 开发目的

此项目旨在为南京航空航天大学住宿生提供更方便的电费查询方式。

本人优先开发怡园21栋空调用电。



## 说明

该项目刚刚起步，目前主要的问题是，电费类目与参数的对应太过复杂。

`src/main.py`是获取电费的python脚本。
`src/index.php`是与微信交互的页面。
`src/wechat.class.php`是瑞辅大佬给的微信库。
`inspect/inspect.txt`是本人暂时整理的对应关系。
